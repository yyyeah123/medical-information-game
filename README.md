# 谣言诊疗室 H5 小游戏

## 运行方式

1. 用 VSCode 打开整个 `rumor-clinic-game` 文件夹。
2. 安装 VSCode 插件：Live Server。
3. 右键 `index.html`，选择 `Open with Live Server`。
4. 浏览器打开后即可试玩。

## 文件说明

- `index.html`：页面结构
- `style.css`：界面样式、诊疗室风格、响应式适配
- `app.js`：10 道题目、判断逻辑、积分、诊断书弹窗、最终评级

## 修改题目

打开 `app.js`，修改最上方 `questions` 数组即可。
